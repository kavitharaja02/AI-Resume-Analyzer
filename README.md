# AI Resume Analyzer 🚀

## 📌 Overview

AI Resume Analyzer is a web application built using Python and Streamlit that analyzes resumes and provides a skill-based score along with suggestions for improvement.

---

## 🎯 Features

* Upload PDF Resume
* Automatic Text Extraction
* Skill Detection
* Resume Scoring System
* Improvement Suggestions
* Simple and Interactive UI

---

## 🛠 Tech Stack

* Python
* Streamlit
* PyPDF2
* Basic NLP Logic

## ▶️ Installation

1. Clone the repository
2. Install dependencies:

pip install -r requirements.txt

3. Run the app:

streamlit run app.py

---

## 📊 How It Works

1. User uploads resume.
2. Text is extracted from PDF.
3. Skills are matched with predefined database.
4. Score is calculated.
5. Suggestions are displayed.
